# Dashboard

- Dashboard that gives a overview of stats.

# Features:-

- Light and Dark mode.
- Grid Tables customizable.
- Visual Charts representations of stats (using Bar, Line, Pie and Geographical Charts).

# Tech Stack Used:-

- React Js.
- Material-UI.
- Nevo Charts.
